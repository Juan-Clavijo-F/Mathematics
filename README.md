# Mathematics

Welcome to my personal mathematics repository.

This project contains a growing collection of mathematical notes written in LaTeX, developed throughout my study of pure mathematics. Its primary purpose is to organize mathematical knowledge in a clear, rigorous, and well-structured way.

The repository currently contains notes on algebra, analysis, number theory, and complementary topics. New subjects will be incorporated as the project continues to grow.

## Objectives

- Develop rigorous mathematical notes.
- Improve mathematical writing and exposition.
- Organize definitions, theorems, proofs, and examples.
- Build a long-term reference for learning and teaching mathematics.

## Repository structure

```
mathematics/
├── algebra/
├── analysis/
├── number-theory/
└── complement/
```
