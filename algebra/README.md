# Algebra

This directory contains my personal notes on algebra, written in LaTeX as part of an ongoing study project.

These notes are primarily intended to support my own learning by organizing mathematical ideas in a clear and rigorous way. By making them publicly available, I hope they may also be useful to other students interested in algebra.

The current material includes:

- Group Theory
- Ring Theory

- Module Theory
