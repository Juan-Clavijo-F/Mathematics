# Number Theory

This directory contains my personal notes on number theory, written in LaTeX as part of an ongoing study project.

These notes are intended to support my own learning by organizing mathematical ideas in a clear and rigorous way. By making them publicly available, I hope they may also be useful to other students interested in number theory.

The collection is currently under development. It is planned to include notes on:

- Elementary Number Theory
- Algebraic Number Theory
- Analytic Number Theory

The first stage of the project focuses on elementary number theory.

---

*These notes are continuously revised and expanded as my understanding of the subject grows.*
